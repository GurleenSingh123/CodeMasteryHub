What are HTML Attributes?

HTML attributes provide additional information about HTML elements. They are always written inside the opening tag.

Basic Syntax:

<tagname attribute="value">Content</tagname>

Example:

<a href="https://example.com">Visit Website</a>

Here:

  • href - Attribute
  • "https://example.com" - Value

Why HTML Attributes Exist

  • Attributes Define Behavior - HTML attributes play a major role in defining the behavior of HTML elements. They determine how an element acts or responds when a user interacts with the interface.
  • Attributes Define Appearance - HTML attributes play an important role in determining how elements or content are displayed on a webpage. While HTML provides the structure of the page, attributes help define presentation details such as fonts, colors, sizes, and layout.
  • Attributes Define Identification - HTML attributes such as id and class help identify elements either uniquely or as part of a group, allowing developers to apply styles and functionality efficiently on a webpage.
  • Attributes Define Accessibility - HTML attributes play an important role in making websites accessible to all users, including people with disabilities. Accessibility means designing websites in such a way that they can be understood, navigated, and interacted with by everyone, regardless of their physical or technical limitations.
  • Attributes Provide Data to Elements - HTML attributes help provide essential data to elements so that they can function correctly. Some elements are incomplete or non-functional without attributes.

Types of Attributes

HTML attribute can be categorized based on their usage and functionality. Each type of attribute has a differnet purpose, functionality, behaviour, styling, indentification, or accessbility.

Understanding these types helps you write clean, organized and professional HTML code.

  • Global Atrributes - Global attributes are not restricted a specific tag and can be used with almost all the HTML elements. This attribute is widely used for styling, indentification, and accessbility.

    • Examples:

    • id - Attribute
      • <p id="intro">Welcome</p>

      The id attribute assign a unique identity to an element.

      Cannot be reprated on the same page.

    • class - Attribute
      • <p class="text">Hello World</p>

      The class attribute can group multiple elements.

      It can be used on the same page.

    • style - Attribute
      • <p style="color: red;">This is a text</p>

      style attribute applies inline css to an element.

  • Element Specific Attribute - These attributes are used with the specific elements only and define the unique functionality.

    • Examples:

    • Anchor Attribute
      • <a href="https://example.com">Visit Link</a>

      href - Tells the destination of the url.

    • Image Attribute
      • <img src="image.jpg" alt="tutorial image"></img>

      src - This tell the image path, where the image is located.

      alt - This tell about the image such as image description.

    • Input Attribute
      • <input type="text" placeholder="Enter your name"></input>

      type - This tell the type of input to be written.

      placeholder - This tell what type of content is to be written in the input.

    • Form Attribute
      • <form action="/submit" method="POST"></form>

      action - This tells where the data will go.

      method - This tells the method of the form is GET or POST.
